Title(B):box-sizing:border-box;


                    box-sizing: border-box; 是 CSS 中一個屬性，它影響元素的盒模型。
                    盒模型是用於定義元素在網頁中的佈局和尺寸的模型，
                    它包括元素的內容、邊框、填充和外邊距。
                    當我將元素的 box-sizing 屬性設置為 border-box 時，元素的
                    寬度和高度將包括元素的內容、填充和邊框。
                    這個屬性的作用在於簡化佈局計算和處理，更容易控制元素的寬度和高度，
                    而不必考慮邊框和填充的影響。
                    舉例來說，假設我有一個具有 width: 200px; 和 padding: 20px; 的元素，
                    在默認的 box-sizing: content-box; 下，元素的寬度將是 240px（200px + 2 * 20px 填充）
                    如果使用 box-sizing: border-box; 那麼元素的寬度將保持為 200px，填充將被包含在內。

Title(B):display:block


                    1.佈局：
                                塊級元素會佔據可用寬度的整個行，並在水平方向上擁有獨立的寬度。它們會從新
                                的一行開始，並在前後都插入額外的空白，形成一個獨立的塊。
                    2.寬度：
                                塊級元素可以設置寬度和高度，並預設擁有 100% 的寬度，即擴展到其容器的寬度。
                    3.高度：
                                塊級元素會在垂直方向上擁有自己的高度，不與其他元素共享行高。
                    4.元素間距：
                                塊級元素之間通常存在空白，因為每個塊狀元素都會開始一個新行。

                    使用的情境包括：

                                 當你想要元素佔據整個可用寬度時。
                                 當你想要元素具有獨立的寬度和高度時。
                                 當你希望元素之間有明顯的區分，形成垂直堆疊的布局時。
                                 總的來說，display: block; 用於塊級元素，它對於構建頁面的結構和佈局非常重要。

Title(D):display:flex + flex: x x x中的三個參數


                    1.第一個參數（flex-grow）：它代表彈性容器中的彈性增長因子。這個值決定了
                        元素在父容器中可用的額外空間中增長的比例。
                        所有子元素的 flex-grow 都是相同的，它們將平均分配可用的空間。
                        某個元素的 flex-grow 值是其他元素的兩倍，它將獲得比其他元素多一倍的可用空間。

                    2.第二個參數（flex-shrink）：這代表彈性容器中的彈性收縮因子。這個值決定了
                        元素在父容器中空間不足時縮小的比例。
                        所有子元素的 flex-shrink 都是相同的，它們將以相同的比例縮小以適應父容器。
                        某個元素的 flex-shrink 值是其他元素的兩倍，那麼它將比其他元素更快地縮小。

                    3.第三個參數（flex-basis）：它代表元素的初始大小。
                    這可以是一個固定的長度（像素、百分比等）或者是 auto。
                    當 flex-basis 設置為 auto 時，元素的初始大小將由其內容的大小決定，
                    而 flex-grow 和 flex-shrink 屬性將根據這個初始大小來分配空間。

                    MEMO:綜合來說，這三個參數共同影響了元素在彈性佈局中的
                                增長、收縮和初始大小，這有助於實現靈活的版面設計。

Title(D):display:inline or inline-block


                    display: inline; 和 display: inline-block;它們之間有一些關鍵的區別。

                    1.佈局模型：
                               display: inline; 使元素呈現為內聯元素，類似於文字的行為。
                               它不接受寬度（width）和高度（height）屬性。水平方向上，元素會盡可能佔據與
                               內容行高相同的空間，而垂直方向上，不會影響行高。
                    2.盒模型：         
                               display: inline; 的盒模型對外表現為一個內聯元素，它不會產生塊狀的盒子，因此水
                               平方向上不會影響其他元素。
                               display: inline-block; 的盒模型對外表現為一個內聯塊元素，它既具有塊級元素的盒
                               模型特性（可以設置寬度和高度），又在水平方向上不會撐滿整個行。
                    3.元素間距：
                               在使用 display: inline; 時，元素之間的空白符（空格、換行等）會導致它們之間存
                               在空白間隙，因為它們被視為文字。
                               display: inline-block; 元素之間的空白符會產生空隙，但你可以通過一些技巧（例如
                               將父元素的 font-size 設置為 0）來解決這個問題。
                               總之，display: inline; 適用於行內元素的佈局，而 display: inline-block; 適用於希望
                               具有塊級元素特性但仍然保持行內元素外觀的情況。

                               總之，display: inline; 適用於行內元素的佈局，而 display: inline-block; 適用於希望
                               具有塊級元素特性但仍然保持行內元素外觀的情況

                    Grid Layout 是 CSS 中的排版方式之一，可以幫助建立一個複雜的網頁佈局。

                    2.定義格線：
                        CSS 中，使用 'grid-template-columns' 和 'grid-template-rows' 屬性來定義格線的結構。
                        可以指定每個格線的大小，
                        可以是固定大小、彈性大小(使用 fr 單位)或自動調整(使用 auto 單位)。

                        .grid-container {
                            display: grid;
                            grid-template-columns: 1fr 2fr 1fr; /* 三列，中間的列寬度是兩側的兩倍 */
                            grid-template-rows: auto 100px; /*兩行，第一行高度自動，第二行高度為100像素*/
                        }
                    =================================================================

                        將子元素放置到格容器內，使用 grid-column 和 grid-row 屬性來指定子元素應該

                        .item1 {

                            grid-row: 1;              /* 子元素在第一行 */
                        }
                        .item2 {
                            grid-cloumn: 2;  /* 子元素在第二列 */
                            grid-row: 2;        /* 子元素在第二行 */
                        }
                    =================================================================
                    4.可選-自動佈局：
                        如果不想手動指定每個子元素的位置，可以使用 grid-auto-flow 屬性來佈局子元素。

                        .grid-container {
                            grid-auto-flow: row;   /* 子元素會一行接一行自動排列 */
                        }
                    =================================================================

                        還可以使用 grid-template-areas 屬性來以名稱指定區域，然後在子元素中

                        }

Title(H):（高度 height）什麼時候適合設為0？


                    1.隱藏元素：
                        希望將一個元素完全隱藏起來時，可以將其高度設置為0，
                        可以使元素的內容和邊框都不可見。
                        ↓↓↓示例如下↓↓↓：
                        .hidden-element {
                            height: 0;
                            overflow: hidden; /*可用於隱藏溢出內容*/
                        }
                    =================================================================
                    2.製作三角形或箭頭：
                        通過將一個塊級元素的寬度和高度設定為 0，
                        並將邊框屬性添加到元素，你可以建立一個三角形或箭頭。
                        ↓↓↓示例如下↓↓↓：
                        .triangle {
                            width: 0;
                            height: 0;
                            border-left: 20px solid transparent;
                            border-right: 20px solid transparent;
                            border-bottom: 20px solid red; /*可以換成其它顏色*/
                        }
                    =================================================================
                    3.清除浮動：
                        我們可能需要在容器內清除浮動以確保容器包含浮動的子元素。
                        設置容器的高度為 0 ，然 使用 clear: both; 屬性來清除浮動。
                        ↓↓↓示例如下↓↓↓：
                        .clearfix::after {
                            content: '';
                            display: block; or table;
                            clear: both;
                            height: 0;
                        }
                    =================================================================
                    4.製作純CSS加載動畫：
                        通過將一個元素的高度設定為 0 ，然後使用動畫或過渡效果來逐漸增加其高度，
                        可以製作純CSS的加載動畫效果。
                        ↓↓↓示例如下↓↓↓：
                        .loading-spinner {
                            height: 0;
                            animation: spin 2s infinite linear;
                        }
                        @keyframes spin {
                            0% {transform: rotate(0deg);}
                            100% {transform; rotate(360deg);}
                        }

Title(W):Width And Height no effect 寬度與高度設定卻沒效果？


                    網頁切版有著這麼一句話流傳：「若寬度與高度設定後沒有效果，不是inline，就是block」
                    1.inline元素:inline元素是指行內元素，如文字、連結、圖片等。
                       行內元素通常不具備寬度和高度，它們會自動適應其內容的大小。
                       如果嘗試為行內元素設定寬度和高度，這些設定可能會被忽略？
                       因為行內元素的特性是在同一行內排列，它們不會占據高立的區域。

                    2.block元素:block元素是指塊級元素，如段落、標題、＜div＞等。
                       塊級元素通常可以設定寬度和高度，因為它們在文檔流中占據獨立的區域，
                       可以在頁面上占用一整行或更多的空間。
                       設定塊級元素的寬度和高度可以明顯地影響它們益頁面中的布局。
                    
                    3.所以當我們在設定元素的寬度和高度後，卻沒看到效果時，首先要檢查
                       該元素是被視為「行內元素」還是「塊級元素」。
                       如果元素被視為行內元素，嘗試將其轉換為塊級元素，可以使用CSS的display:block;
                       將行內元素改成為塊級元素，然後設定寬度和高度。
